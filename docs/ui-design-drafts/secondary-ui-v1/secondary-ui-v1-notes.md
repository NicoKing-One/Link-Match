# Link Match 二级页 UI 设计稿说明

生成日期：2026-06-15

## 1. 本轮交付

本轮只生成水果森林主题的二级页高保真设计稿，不改代码、不接入主游戏。

文件：

- `profile-fruit-forest.png`：个人中心页。
- `settings-fruit-forest.png`：设置页。
- `exchange-fruit-forest.png`：金币兑换页。

三张图均以现有首页设计稿 `home-ui-design-v2-fruit-forest.png` 为视觉母版，延续青绿色果冻按钮、奶油卡片、白色描边、柔和高光、森林水果边缘装饰和童话感背景。

## 2. 页面用途

### 个人中心

用于首页左上角“个人”入口。

建议动态字段：

- 玩家昵称。
- 玩家 ID。
- 当前进度。
- 总星数。
- 金币数量。
- 已通关数量。
- 三星关卡数量。

可切图元素：

- 背景装饰。
- 顶部标题牌。
- 返回按钮底图。
- 头像框。
- 奶油信息卡底图。
- 底部“返回首页”按钮底图。

### 设置页

用于首页右上角“设置”入口。

建议动态字段：

- 音乐开关状态。
- 音效开关状态。
- 振动开关状态。

可切图元素：

- 背景装饰。
- 顶部标题牌。
- 返回按钮底图。
- 设置面板底图。
- 开关底图。
- “清除进度”按钮底图。
- 底部“返回首页”按钮底图。

### 金币兑换页

用于后续金币资源系统入口。

建议动态字段：

- 当前金币数量。
- 体力兑换数量。
- 提示兑换数量。
- 洗牌兑换数量。
- 各兑换项金币价格。

可切图元素：

- 背景装饰。
- 顶部标题牌。
- 返回按钮底图。
- 金币资源卡底图。
- 兑换卡片底图。
- 体力、提示、洗牌图标。
- “兑换”按钮底图。
- 底部“返回首页”按钮底图。

## 3. 接入建议

- 不建议把整张设计稿直接贴进游戏。
- 中文文案、数字、资源数量、开关状态应优先保留在 HTML/CSS 层。
- 背景、卡片底图、按钮底图和图标可以从设计稿中拆成可复用资源。
- 糖果花园和果冻城堡后续应沿用同一页面结构，只替换主题色、边缘装饰和局部图标质感。
