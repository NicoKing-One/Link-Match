# 设置页面资源

页面入口：首页右上角“设置”按钮。

本文件夹只放设置页面需要调用的图片和图标。接入时页面可以统一从：

```text
src/assets/UI-ICON/settings-page/
```

读取资源。

## 按钮

- `button-back-icon.png`：顶部返回按钮。
- `button-return-home-bg.png`：底部“返回首页”按钮背景图，文字建议用 HTML/CSS 覆盖。
- `switch-on-icon.png`：开启状态开关图标。

## 全屏背景

- `page-bg.png`：页面全屏背景图。三个二级页当前统一使用这一张背景。

## 顶部标题

- `title-bg-settings.png`：顶部“设置”标题背景图，文字建议用 HTML/CSS 覆盖。

## 主体模块

- `module-settings-panel-bg.png`：设置主面板背景。
- `module-settings-row-bg.png`：音乐、音效、振动单行设置背景。
- `module-settings-clear-bg.png`：清除进度区域背景。

## 图标

- `icon-music.png`：音乐图标。
- `icon-sound.png`：音效图标。
- `icon-vibration.png`：振动图标。
